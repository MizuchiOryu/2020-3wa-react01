# GESTIOND DES STYLES

## Notions nécessaires
- terminal
- npm >= 5.6
- nodeJS  >= 8.10

## Nouvelles notions
- package.json
- Build /dev env
- [create-react-app](https://github.com/facebook/create-react-app)


## Codes a tester

```

```

```

```
